## Git convention
- ```Feat : ~ (00%)``` : 개발한 기능을 Commit 할 때 사용합니다.
   - 진행률은 선택사항입니다.
- ```Fix : ~ (00%)``` : 기능이나 타 요소에 대한 수정이 진행되었을 때 사용합니다.
   - 진행률은 선택사항입니다.
- ```Docs : ~ ``` : 문서(README 등의 markdown 를 수정한 경우에 사용합니다.
